# Svelte & Snowpack template

```bash
## Install template
$ npx degit ParkYoungWoong/svelte-snowpack-template DIR_NAME

## Change directory
$ cd DIR_NAME

## Install dependencies
$ npm i

## Start dev server
$ npm run dev
```

- Svelte
- Snowpack
- TypeScript
- SCSS
- Autoprefixer/PostCSS
- Web test runner
- Chai
- Reset.css
